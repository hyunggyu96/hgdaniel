# 🤖 MCP (Model Context Protocol) 가이드

Antigravity가 사용자님의 인프라를 더 강력하게 제어하고 모니터링할 수 있도록 MCP 연동을 도와주는 가이드입니다.

## 1. Firebase MCP 활성화 (필수)

Antigravity가 Firebase 기능을 사용하여 앱 알림, 사용자 관리 등을 하려면 로그인이 필요합니다.

1. 터미널을 열고 아래 명령어를 입력하세요:

   ```bash
   firebase login
   ```

2. 웹 브라우저가 열리면 사장님의 구글 계정으로 로그인하고 권한을 승인하세요.
3. 로그인이 완료되면 저에게 **"Firebase 로그인 완료"**라고 말씀해 주세요.

## 2. Google Cloud (Cloud Run) 활성화

서버 상태를 실시간으로 감시하거나 복잡한 작업을 처리할 때 사용합니다.

1. Google Cloud SDK가 설치되어 있다면 아래 명령어를 입력하세요:

   ```bash
   gcloud auth application-default login
   ```

2. 승인이 완료되면 제가 Google Cloud의 수백 가지 도구에 접근하여 사장님의 프로젝트를 직접 배포하고 관리할 수 있게 됩니다.

---

### 🔥 MCP가 연결되면 할 수 있는 일들

* **실시간 사이트 감시**: 사이트가 멈추면 제가 즉시 인지하고 복구 작업을 시도합니다.
* **자동 뉴스 푸시**: 중요한 뉴스가 뜨는 즉시 파이어베이스를 통해 사장님 스마트폰으로 알림을 보냅니다.
* **로그 정밀 분석**: 서버 에러 로그를 제가 실시간으로 읽고 원인을 찾아 해결합니다.

이 가이드를 따라 준비가 되시면 말씀해 주세요! 제가 바로 다음 단계의 고도화된 자동화를 진행하겠습니다.
