# 📰 Hybrid AI News Dashboard

**Market Intelligence Platform powered by Local AI**

이 프로젝트는 태블릿(Android Termux) 내의 로컬 AI가 24/7 뉴스를 수집/분석하고, Next.js 웹사이트가 이를 시각화하는 하이브리드 시스템입니다.
